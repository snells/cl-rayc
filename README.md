# cl-rayc
simple raycasting with cepl


Depends on [cepl](https://github.com/cbaggers/cepl)

To run clone this project to your quicklisp/local-packages   
(ql:quickload :cl-rayc)   
(in-pacakge :cepl)   
(repl)   
(run-loop)   


w move forward   
s move backward   
a decrease rotation   
d increase rotation
